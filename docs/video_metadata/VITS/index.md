# Vertical Interval Test Signals

This section describes Vertical Interval Test Signals (VITS) used in NTSC 525-line and PAL 625-line analogue video systems.

The waveform diagrams are generated from YAML definitions stored outside the documentation tree at `resources/definitions/vits/`. To recreate the diagrams and refresh the Markdown image links, run:

```sh
nix develop path:. -c scripts/generate-vits-diagrams
```
